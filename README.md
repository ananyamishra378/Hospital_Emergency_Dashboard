
# 🏥 Hospital Emergency Dashboard

## 📊 Project Overview
This project presents an interactive Excel dashboard designed to analyze hospital emergency room data. It focuses on patient flow, waiting time, service efficiency, and department referrals to provide meaningful healthcare insights.

---

## 🎯 Objectives
- To analyze the number of patients visiting the hospital
- To evaluate average waiting time
- To identify delays in patient service
- To understand department-wise referrals
- To analyze patient distribution by age and gender

---

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Data Visualization Techniques
- Dashboard Design

---

## 📈 Key Metrics
- Total Patients: 4338+
- Average Wait Time: ~35 minutes
- Patient Satisfaction Score: 4.99

---

## 🔍 Key Insights
- 59% patients were attended on time, while 41% experienced delays
- Majority of patients fall in the 20–40 age group
- General Practice received the highest number of referrals
- Gender distribution is nearly balanced
- Patient flow remains consistent with slight monthly variations

---

## 📊 Dashboard Features
- Monthly filter for dynamic analysis
- Patient distribution by age group
- Admission status tracking
- Department-wise referral analysis
- Gender-wise comparison
- Visual charts for better decision-making

---

## 💡 Conclusion
This dashboard helps in identifying operational inefficiencies and supports better decision-making in hospital management by providing clear and actionable insights.
